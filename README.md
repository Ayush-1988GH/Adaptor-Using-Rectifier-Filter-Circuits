# Adaptor-Using-Rectifier-Filter-Circuits
12V Regulated DC Bench Supply from 220V AC using 12-0-12V Centre Tapped Transformer

### 12V Regulated DC Power Supply using Full Wave Rectifier

Designed and implemented a **center-tapped full wave rectifier** with filtering and **Zener diode regulation** to obtain a stable **~12 V DC output**. The circuit converts AC to DC using **1N4007 diode** rectification, smoothens ripple using a **2200 µF capacitor**, and regulates voltage using a Zener diode.

The setup functions as a **mini bench power supply**, capable of delivering approximately **100–120 mA**, with:

* LED load demonstration
* Voltage display module
* Protected test output for components

This project demonstrates key concepts of **rectification, filtering, voltage regulation, and load handling** in basic power electronics.
